On the island Sophie found the interesting round amulet. Along the perimeter of the amulet has a small points through each degree (total 360). Also the amulet has three levers with different lengths (first is the shortest, second is middle and third is the longest). Levers can be rotated. If you rotate one lever, then rotate and others. The levers rotate discretely by degrees. Clockwise direction is a positive angle. Counterclockwise is a negative.

After some experiments Sophie discovers the levers correlation. If you rotate a lever, then other rotate in the same direction but with different speed.
If you rotate the first, then the second is rotated at the f2 time faster, the third -- at the f3time faster.
If you rotate the second, then the third is rotated at the s3 time faster, the first -- at the s1 time faster.
If you rotate the third, then the first is rotated at the t1 time faster, the second -- at the t2 time faster.
This correlation can be represented as a matrix 3x3 with 1 in the main diagonal.
1 f2 f3
s1 1 s3
t1 t2 1
For example, the rotating correlation is given as the matrix:
1 2 3
3 1 2
2 3 1
This mean that:
If you rotate the first, the second is rotated at the double angle while the third rotates at triple.
If you rotate the second, the third is rotated at the double angle, and the first rotates at triple.
If you rotate the third, the first is rotated at the double angle, and the second rotates at triple.
For example: You rotate the first lever at +10 degrees. As a result, the second is rotated at +20 degrees and the third at +30 degrees.

You can rotate the levers from -180 to +180 degrees only (but in interrelated mode it can be rotated at more than 360). You can rotate each lever once in the next order: the first, the second, the third. Sophie used the X-ray and found out what position it should be. After these rotations the levers must be at 0, 225, 315.

You are given a rotation matrix as a list of lists with positive integers and the unitary main diagonal. You should return a list of three integers -- the angles for each lever. The start position of the levers is [0, 0, 0].

Input: A rotation matrix, a list of lists.

Output: The angles, a list of integers.

Example: 

checkio(90, 0, 180)
checkio(0, 0, 0)
